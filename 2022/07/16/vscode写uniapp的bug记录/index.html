vscode写uniapp的bug记录

新建uniapp组件

vscode在生成新组件时，结构是不完整的，script 中生成的的 export default { 模块少了一个 “ } ”，这算是vscode的一个漏洞叭

UI组件：uni-swipe-action-item

这个ui组件的问题可以说是折磨的ZOE很惨很惨（>_<），这是一个实现滑动删除功能的组件，如下图

使用这个组件，报的错误很离谱，搜了很多网站都是答非所问，好在ZOE最后都解决啦~ヾ(✿ﾟ▽ﾟ)ノ

TypeError: Cannot read property ‘userAgent’ of undefined - Search

  • 这个ZOE跟着各种各样、五花八门的博客试了好多种方法

  • 最后是改了 项目名称\node_modules@dcloudio\uni-ui\lib\uni-swipe-action-item\mpxs.js 中的代码

  • 这样改的原因是： is_pc 是H5特有的，在小程序中使用会报错

TypeError: uni.requireNativePlugin is not a function - Search

  • ZOE也不清楚为什么要这样改的，但确实是有用的

TypeError: this.swipeaction.closeOther is not a function
